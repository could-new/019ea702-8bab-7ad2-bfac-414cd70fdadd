# Karaoke App

Кроссплатформенное приложение для исполнения караоке, созданное с помощью Flutter.

## Возможности
- **База песен**: Выбор песен из каталога.
- **Запись**: Запись вашего исполнения под фонограмму.
- **Редактирование и Мастеринг**: Настройка громкости голоса, добавление эффектов (реверберация, эхо) после записи.

## Запуск
Для запуска приложения используйте стандартные команды Flutter:
```bash
flutter pub get
flutter run
```

## Технологии
- Flutter / Dart
- Material Design 3
